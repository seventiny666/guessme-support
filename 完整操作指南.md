# 完整操作指南 - 部署到 guessme-support

## 🎯 目标
将技术支持页面部署到您的公开仓库并启用GitHub Pages

## 📋 准备工作
您需要以下4个文件的内容（已在docs/目录中准备好）：
- index.html - 技术支持页面
- privacy.html - 隐私政策
- terms.html - 使用条款  
- README.md - 项目说明

## 🚀 操作步骤

### 第1步：打开仓库
访问：https://github.com/seventiny666/guessme-support

### 第2步：上传文件
对每个文件重复以下步骤：

1. 点击 "Add file" → "Create new file"
2. 输入文件名
3. 复制对应文件内容
4. 点击 "Commit changes"

### 第3步：启用GitHub Pages
1. 点击 "Settings"
2. 左侧菜单找到 "Pages"
3. Source: "Deploy from a branch"
4. Branch: "main"
5. Folder: "/"
6. 点击 "Save"

### 第4步：验证
等待1-5分钟后访问URL验证

## 📄 文件内容
所有文件内容都在本项目的docs/目录中，可直接复制使用。
