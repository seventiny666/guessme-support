# iOS 自动续订订阅功能配置指南

## 📋 功能概述

本项目实现了完整的iOS自动续订订阅功能，支持：
- ✅ 4种订阅类型：周订阅、月订阅、年订阅、终身版
- ✅ 自动续费管理
- ✅ 购买验证和状态持久化
- ✅ 恢复购买功能
- ✅ 沙盒测试支持
- ✅ 超时处理和错误管理
- ✅ 多语言支持
- ✅ 合规的订阅说明文案

## 🛠 App Store Connect 配置步骤

### 1. 创建订阅群组
1. 登录 [App Store Connect](https://appstoreconnect.apple.com)
2. 选择你的应用 → **功能** → **App 内购买项目**
3. 点击 **+** 创建订阅群组
4. 群组名称：`Pro Subscription`
5. 参考名称：`专业版订阅`

### 2. 创建自动续期订阅产品

#### 周订阅 (Weekly)
- **产品ID**: `com.seventinygame.app.weekly`
- **参考名称**: `Pro Weekly Subscription`
- **订阅时长**: 1周
- **价格层级**: $1.99
- **免费试用**: 3天
- **显示名称**: 专业版周订阅
- **描述**: 解锁所有游戏分类。前3天免费试用，之后每周自动续订。

#### 月订阅 (Monthly)
- **产品ID**: `com.seventinygame.app.monthly`
- **参考名称**: `Pro Monthly Subscription`
- **订阅时长**: 1个月
- **价格层级**: $3.99
- **免费试用**: 无
- **显示名称**: 专业版月订阅
- **描述**: 解锁所有游戏分类，每月自动续订。

#### 年订阅 (Yearly) - 推荐
- **产品ID**: `com.seventinygame.app.yearly`
- **参考名称**: `Pro Yearly Subscription`
- **订阅时长**: 1年
- **价格层级**: $19.99
- **免费试用**: 3天
- **显示名称**: 专业版年度订阅
- **描述**: 解锁所有游戏分类，享受完整游戏体验。前3天免费试用，之后每年自动续订。最超值选择！

#### 终身版 (Lifetime)
- **产品ID**: `com.seventinygame.app.lifetime`
- **参考名称**: `Pro Lifetime`
- **产品类型**: 非续期订阅
- **价格层级**: $24.99
- **显示名称**: 专业版终身版
- **描述**: 一次性购买，永久解锁所有专业版功能。无需订阅，买断制使用。

### 3. 配置订阅群组层级
确保订阅产品按以下顺序排列（从低到高）：
1. 周订阅 (Group Number: 1)
2. 月订阅 (Group Number: 2)  
3. 年订阅 (Group Number: 3)

## 🧪 沙盒测试配置

### 1. 创建沙盒测试账号
1. App Store Connect → **用户和访问** → **沙盒测试员**
2. 点击 **+** 创建测试账号
3. 填写测试邮箱和密码
4. 选择地区（建议选择美国）

### 2. 设备配置
1. 在测试设备上退出App Store账号
2. 在应用内触发购买时，系统会提示登录
3. 使用沙盒测试账号登录
4. 完成测试购买

### 3. 沙盒测试特点
- 订阅会快速过期便于测试：
  - 1周订阅 → 3分钟过期
  - 1月订阅 → 5分钟过期
  - 1年订阅 → 1小时过期
- 免费试用期缩短为几分钟
- 可以测试订阅升级/降级
- 支持恢复购买测试

## 💻 代码集成说明

### 1. StoreManager 使用
```swift
// 获取StoreManager实例
let storeManager = StoreManager.shared

// 检查订阅状态
if storeManager.hasProVersion {
    // 用户已订阅专业版
}

// 购买产品
if let product = storeManager.getProduct(for: .yearlySubscription) {
    do {
        let transaction = try await storeManager.purchase(product)
        // 购买成功
    } catch {
        // 处理购买错误
    }
}

// 恢复购买
await storeManager.restorePurchases()
```

### 2. 订阅状态监听
StoreManager 使用 `@Published` 属性，可以在 SwiftUI 中自动更新UI：
```swift
struct ContentView: View {
    @StateObject private var storeManager = StoreManager.shared
    
    var body: some View {
        if storeManager.hasProVersion {
            PremiumContentView()
        } else {
            FreeContentView()
        }
    }
}
```

### 3. 错误处理
```swift
// 监听错误状态
.alert("错误", isPresented: .constant(storeManager.errorMessage != nil)) {
    Button("确定") {
        storeManager.errorMessage = nil
    }
} message: {
    if let errorMessage = storeManager.errorMessage {
        Text(errorMessage)
    }
}
```

## ⚠️ 常见问题

### Q: 产品加载失败怎么办？
A: 检查以下几点：
1. 产品ID是否与App Store Connect中完全一致
2. 应用是否已在App Store Connect中创建
3. 订阅产品状态是否为"准备提交"
4. 网络连接是否正常

### Q: 沙盒测试购买失败？
A: 确认：
1. 使用的是沙盒测试账号
2. 设备上已退出正式App Store账号
3. 应用使用开发证书构建
4. Products.storekit文件配置正确

### Q: 订阅状态不同步？
A: StoreManager会自动处理状态同步，如果遇到问题：
1. 调用 `await storeManager.restorePurchases()`
2. 检查网络连接
3. 重启应用

### Q: 如何测试订阅过期？
A: 在沙盒环境中：
1. 购买订阅后等待自动过期
2. 观察应用状态变化
3. 测试过期后的功能限制
