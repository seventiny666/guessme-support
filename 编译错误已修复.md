## 常见问题

### Q: 还是提示找不到 LanguageManager
A: 
1. 关闭 Xcode
2. 删除 DerivedData：`rm -rf ~/Library/Developer/Xcode/DerivedData/*`
3. 重新打开 Xcode
4. Clean Build Folder (Shift + Cmd + K)
5. 重新编译 (Cmd + B)

### Q: 文件显示红色
A: 
1. 在 Xcode 中右键点击文件
2. 选择 "Show in Finder"
3. 确认文件存在
4. 如果不存在，重新添加文件

### Q: 编译警告
A: 
- 检查是否有未使用的变量
- 检查是否有废弃的 API
- 根据警告提示修复
