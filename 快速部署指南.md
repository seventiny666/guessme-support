# 快速部署指南 - guessme-support 仓库

## 🎯 目标
将技术支持、隐私政策和使用条款页面推送到您的公开仓库，并启用GitHub Pages。

## ⚡ 快速步骤

### 1️⃣ 打开您的仓库
https://github.com/seventiny666/guessme-support

### 2️⃣ 上传4个文件

#### 文件1: index.html（技术支持页面）
- 点击 "Add file" → "Create new file"
- 文件名: `index.html`
- 内容: 复制 `docs/index.html` 的全部内容
- 提交

#### 文件2: privacy.html（隐私政策）
- 点击 "Add file" → "Create new file"
- 文件名: `privacy.html`
- 内容: 复制 `docs/privacy.html` 的全部内容
- 提交

#### 文件3: terms.html（使用条款）
- 点击 "Add file" → "Create new file"
- 文件名: `terms.html`
- 内容: 复制 `docs/terms.html` 的全部内容
- 提交

#### 文件4: README.md（项目说明）
- 点击 "Add file" → "Create new file"
- 文件名: `README.md`
- 内容: 复制 `docs/README.md` 的全部内容
- 提交

### 3️⃣ 启用 GitHub Pages
1. 点击 "Settings"
2. 左侧菜单找到 "Pages"
3. Source: 选择 "Deploy from a branch"
4. Branch: 选择 "main"
5. Folder: 选择 "/" (root)
6. 点击 "Save"

### 4️⃣ 等待部署
- 通常需要 1-5 分钟
- 看到绿色提示表示成功

### 5️⃣ 验证URL
访问以下地址验证：
- https://seventiny666.github.io/guessme-support/
- https://seventiny666.github.io/guessme-support/privacy.html
- https://seventiny666.github.io/guessme-support/terms.html

### 6️⃣ 在 App Store Connect 中填写

**隐私政策网址**:
```
https://seventiny666.github.io/guessme-support/privacy.html
```

**技术支持网址**:
```
https://seventiny666.github.io/guessme-support/
```

## ✅ 完成！

现在您的应用就有了公开的技术支持和隐私政策页面，可以提交到 App Store 审核了。

---

## 📝 需要复制的文件内容

所有文件内容都在本项目的 `docs/` 文件夹中：
- `docs/index.html` - 技术支持页面
- `docs/privacy.html` - 隐私政策
- `docs/terms.html` - 使用条款
- `docs/README.md` - 项目说明

## 🆘 遇到问题？

### 问题1: 页面显示404
- 确保已点击 "Save" 按钮
- 等待 5-10 分钟
- 清除浏览器缓存
- 在隐私浏览模式下尝试

### 问题2: 文件上传失败
- 检查网络连接
- 确保文件名正确（小写）
- 尝试重新上传

### 问题3: GitHub Pages 没有启用
- 确保选择了正确的分支（main）
- 确保选择了正确的文件夹（/）
- 点击 "Save" 后等待部署
