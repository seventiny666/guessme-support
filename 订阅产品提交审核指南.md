# 订阅产品提交审核指南

## 问题描述
```
Guideline 2.1(b) - Performance - App Completeness
We are unable to complete the review of the app because one or more of the In-App Purchase products have not been submitted for review
```

## 问题原因
App 中使用了订阅功能，但订阅产品没有提交审核。Apple 要求 App 和其内购产品必须同时审核。

## 解决步骤

### 第一步：访问订阅配置页面

1. 登录 App Store Connect
2. 我的 App -> GuessMe -> 功能 -> App 内购买项目
3. 或直接访问：https://appstoreconnect.apple.com/apps/6759666661/distribution/subscription-groups

### 第二步：检查订阅产品状态

找到你的订阅产品：`com.seventinygame.app.yearly`

**当前状态可能是：**
- 🟡 "准备提交" - 需要提交审核
- 🟡 "元数据丢失" - 需要完善信息
- 🔴 "被拒绝" - 需要修复问题

### 第三步：完善订阅信息（如果需要）

#### 3.1 订阅群组本地化
点击订阅群组 -> 本地化 -> 添加本地化

**简体中文：**
- 订阅群组显示名称：`专业版订阅`
- 自定义 App 名称：`你比划我猜`

#### 3.2 订阅产品本地化
点击订阅产品 -> 订阅本地化 -> 添加本地化

**简体中文：**
- 显示名称：`专业版年度订阅`
- 描述：
```
解锁全部功能，享受完整游戏体验

包含内容：
• 10+ 精选游戏分类
• 700+ 精选词汇库
• 创建自定义分类
• 无限制游戏次数
• 后续版本免费升级

订阅说明：
• 前3天免费试用
• 试用期结束后自动续订 $1.99/年
• 可随时在设置中取消订阅
• 取消后仍可使用至订阅期结束
```

#### 3.3 审核信息
点击订阅产品 -> 审核信息

**订阅审核截图：**
- 上传订阅购买界面的截图
- 建议尺寸：iPhone 14 Pro Max (1290 x 2796)
- 截图内容：PurchaseView 界面

**审核说明：**
```
订阅测试说明：
1. 在首页点击"升级到专业版"按钮
2. 查看订阅详情和价格信息
3. 点击"开始免费试用"进行购买
4. 验证专业版功能已解锁（所有分类可用）
5. 验证自定义分类功能可用

测试账号：
请使用 App Store Connect 提供的沙盒测试账号进行测试

注意事项：
- 订阅在沙盒环境中会加速（3天试用变为3分钟）
- 可以在 iOS 设置 -> Apple ID -> 订阅中管理测试订阅
- 专业版解锁后，所有10个分类和自定义功能都应该可用
```

### 第四步：提交订阅产品审核

#### 4.1 检查必填信息
确保以下信息已完整：
- [ ] 产品 ID：com.seventinygame.app.yearly
- [ ] 参考名称：Pro Yearly Subscription
- [ ] 订阅时长：1 年
- [ ] 价格：$1.99
- [ ] 免费试用：3 天
- [ ] 简体中文本地化已完成
- [ ] 审核截图已上传
- [ ] 审核说明已填写

#### 4.2 提交审核
1. 在订阅产品页面，点击右上角"提交以供审核"
2. 确认所有信息正确
3. 点击"提交"

### 第五步：重新提交 App

订阅产品提交后：

1. 返回 App 页面
2. App Store -> 准备提交
3. 在"App 内购买项目和订阅"部分
4. 确认显示你的订阅产品
5. 重新提交 App 审核

## 审核时间线

### 订阅产品审核：
- 提交后：立即进入"等待审核"状态
- 审核时间：通常 24-48 小时
- 结果：通过后状态变为"已批准"

### App 审核：
- 订阅产品通过后，App 才能继续审核
- App 审核时间：通常 24-48 小时
- 总时间：2-4 天

## 注意事项

### 重要提醒：
1. ⚠️ 订阅产品必须先通过审核，App 才能通过
2. ⚠️ 订阅信息修改后需要重新审核
3. ⚠️ 确保 App 代码中的产品 ID 与 App Store Connect 一致
4. ⚠️ 测试时使用沙盒账号，不要使用真实付费

### 产品 ID 验证：
检查以下文件中的产品 ID 是否一致：
- App Store Connect：`com.seventinygame.app.yearly`
- Products.storekit：`com.seventinygame.app.yearly`
- StoreManager.swift：`com.seventinygame.app.yearly`

## 常见错误和解决方案

### 错误1：订阅产品找不到
**原因：** 产品 ID 不匹配  
**解决：** 检查代码中的产品 ID

### 错误2：订阅界面显示错误
**原因：** StoreKit 配置问题  
**解决：** 检查 Products.storekit 文件

### 错误3：无法购买订阅
**原因：** 沙盒账号问题  
**解决：** 使用正确的沙盒测试账号

### 错误4：订阅状态不更新
**原因：** StoreManager 逻辑问题  
**解决：** 检查订阅状态检测代码
