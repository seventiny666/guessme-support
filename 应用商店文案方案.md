# 应用商店文案方案

## 方案1：聚会场景（推荐）
```
主标题：你比划我猜
副标题：聚会必备的比划猜词游戏
```
**特点**：直接点明使用场景，强调"必备"增加吸引力

---

## 方案2：功能导向
```
主标题：你比划我猜
副标题：500+词汇的比划猜词游戏
```
**特点**：突出词汇量优势，吸引重度玩家

---

## 方案3：社交属性
```
主标题：你比划我猜
副标题：多人聚会派对猜词游戏
```
**特点**：强调多人互动和派对属性，适合年轻用户

---

## 方案4：简洁直接
```
主标题：你比划我猜
副标题：经典比划猜词聚会游戏
```
**特点**：用"经典"增加信任感，简洁易懂

## 推荐使用

**中文市场**：方案1 - 聚会必备的比划猜词游戏
- 理由：明确使用场景，"必备"增加下载动力

**英文市场**：方案1 - Party Charades Game
- 理由：Charades是国际通用词汇，Party突出社交属性

---

## 注意事项

1. **副标题限制**：最多30个字符（中文约15个字，英文约30个字母）
2. **关键词覆盖**：副标题中的词会被搜索索引，要包含核心关键词
3. **避免堆砌**：不要过度使用关键词，保持自然流畅
4. **突出差异**：强调你的应用与其他猜词游戏的不同之处

---

## A/B测试建议

如果后续想优化，可以尝试：
- 测试"聚会"vs"派对"哪个转化率更高
- 测试是否需要强调"500+词汇"
- 测试"必备"vs"经典"哪个更吸引人

当前建议先使用方案1，观察数据后再优化。
