常见问题

❓ 如何开始游戏？

1. 在主界面选择一个分类
2. 选择游戏时长（30秒/60秒/90秒）
3. 点击"开始游戏"
4. 一人看屏幕比划，另一人猜词
5. 猜对点击✓，不会点击跳过

💎 如何订阅专业版？

1. 在主界面点击"升级到专业版"按钮
2. 查看订阅详情
3. 点击"开始免费试用"
4. 前3天免费，之后每年1.99美元自动续订

🔄 如何恢复购买？

1. 打开订阅界面
2. 点击底部"恢复购买"按钮
3. 系统会自动检查并恢复您的订阅

❌ 如何取消订阅？

1. 打开iPhone"设置"
2. 点击顶部您的Apple ID
3. 选择"订阅"
4. 找到"你比划我猜"
5. 点击"取消订阅"

注意：取消后，订阅将在当前计费周期结束时停止，期间仍可使用专业版功能。

👨‍👩‍👧‍👦 是否支持家庭共享？

是的！订阅支持家庭共享功能。订阅后，您的家庭成员也可以免费使用专业版功能。

🔐 应用是否收集个人信息？

不会！本应用完全离线运行，不收集任何个人信息，不追踪用户行为，所有数据仅存储在您的设备本地。

📶 需要网络连接吗？

不需要！应用完全离线运行，无需网络连接即可畅玩所有功能。

🆓 免费版和专业版有什么区别？

免费版：可以使用3个分类（随机、成语、美食）
专业版：解锁全部10个分类，包括常识、流行语、动物、名著、网络用语、职业、动画角色
