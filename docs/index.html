❓ Frequently Asked Questions

❓ How to start a game?

1. Select a category on the main screen
2. Choose game duration (30s/60s/90s/120s)
3. Tap "Start Game"
4. One person acts out while another guesses
5. Tap ✓ for correct answers, skip if needed

🔒 Is my data safe?

Absolutely! GuessMe runs completely offline and doesn't collect any personal information. All data is stored locally on your device only.

📱 Which devices are supported?

GuessMe works on iPhone and iPad running iOS 15.0 or later. The app is optimized for all screen sizes.

Yes! GuessMe supports multiple languages including English, Chinese, Japanese, and more.
