# iPad测试验证清单 - 2026年3月11日

## 📋 测试前准备

### 必要步骤 ⚠️
1. **清理Xcode缓存**
   ```bash
   # 清理DerivedData
   rm -rf ~/Library/Developer/Xcode/DerivedData
   
   # 清理模拟器
   xcrun simctl erase all
   ```

2. **删除设备上的应用**
   - 在iPad上长按应用图标
   - 选择"删除App"
   - 确认删除

3. **重新编译安装**
   - 在Xcode中Clean Build Folder (⌘+Shift+K)
   - 重新运行到iPad设备 (⌘+R)
