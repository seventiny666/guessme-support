## 📋 新功能详情

### 自定义分类创建流程

1. **输入分类名称**
   - 例如：水果、动物、品牌等
   - 必填项

2. **输入词汇**
   - 支持逗号、中文逗号、换行分隔
   - 至少5个词汇
   - 实时显示词汇数量

3. **保存分类**
   - 点击"保存分类"按钮
   - 自动保存到本地
   - 显示保存成功提示

4. **使用自定义分类**
   - 保存后在主界面显示
   - 特殊样式：渐变背景 + 橙色边框
   - 点击后选择时长开始游戏

### 自定义分类管理

**已保存的分类列表**
- 显示分类名称和词汇数量
- 支持删除功能
- 数据持久化保存

**分类卡片样式**
- 渐变背景（紫色渐变）
- 橙色边框
- 星星图标 ⭐️
- 与系统分类区分明显

```swift

    id: UUID          // 唯一标识
    name: String      // 分类名称
```

## 🎮 游戏流程

### 系统分类
1. 点击分类卡片
2. 选择游戏时长
3. 开始游戏
4. 游戏中可退出（圆形叉号按钮）

### 自定义分类
1. 点击"创建类型"按钮
2. 输入分类名称和词汇
3. 保存分类
4. 在主界面点击自定义分类
5. 选择游戏时长
6. 开始游戏
7. 游戏中可退出（圆形叉号按钮）
