# 多语言测试指南

## 快速测试步骤

### 1. 添加文件到 Xcode
```
1. 打开 Xcode 项目
2. 右键点击 GuessMe 文件夹
3. 选择 "Add Files to GuessMeIos"
4. 选择 LanguageManager.swift
5. 确保 Target 勾选了 GuessMeIos
6. 点击 Add
```

### 2. 编译运行
```
1. 选择模拟器（iPhone 14 Pro）
2. 按 Cmd + B 编译
3. 按 Cmd + R 运行
```

### 3. 测试语言切换
```
1. 点击首页左上角的国旗按钮（🇨🇳）
2. 在弹出的语言选择器中选择不同语言
3. 观察界面文本是否切换
4. 关闭 App 重新打开，验证语言是否保存
```

## 常见问题排查

### 问题1：编译错误
**检查：**
- LanguageManager.swift 是否添加到项目
- Target 是否正确勾选
- import SwiftUI 是否存在

### 问题2：语言不切换
**检查：**
- @StateObject 是否正确使用
- languageManager 是否初始化
- localizedString 是否正确调用

### 问题3：文本显示 key
**检查：**
- LocalizedStrings 中是否有对应的 key
- 所有语言是否都有该 key
- key 拼写是否正确

### 问题4：国旗不显示
**检查：**
- emoji 是否正确
- 字体大小是否合适
- 模拟器是否支持 emoji
