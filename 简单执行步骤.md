# 🚀 最简单的提交步骤

## 一键执行（推荐）

在项目根目录执行以下命令：

```bash
# 给脚本执行权限
chmod +x quick_deploy_guessme_support.sh

# 执行脚本
./quick_deploy_guessme_support.sh
```

## 手动执行（备选）

如果脚本执行失败，可以手动执行：

```bash
# 1. 创建临时目录并克隆仓库
TEMP_DIR="/tmp/guessme-support-$(date +%s)"
git clone https://github.com/seventiny666/guessme-support.git "$TEMP_DIR"
cd "$TEMP_DIR"

# 2. 清空现有文件
find . -type f -not -path './.git/*' -delete

# 3. 复制新文件（请根据你的实际路径调整）
cp ../docs/index.html ./index.html
cp ../docs/privacy.html ./privacy.html
cp ../docs/terms.html ./terms.html
cp ../docs/README.md ./README.md

# 4. 提交并推送
git add .
git commit -m "更新为英文版本支持页面

- 所有内容翻译为英文
- 邮箱更新为: seventiny007@126.com
- 游戏时长增加120秒选项"

git push origin main

# 5. 清理
cd ..
rm -rf "$TEMP_DIR"
```
