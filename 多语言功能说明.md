## 使用方法

### 用户操作
1. 点击首页左上角的国旗按钮
2. 在弹出的语言选择器中选择语言
3. 界面立即切换到选中的语言
4. 语言偏好自动保存

### 开发者使用
```swift
// 获取本地化文本
languageManager.localizedString("app_title")

// 切换语言
languageManager.currentLanguage = .english

// 当前语言
let current = languageManager.currentLanguage
```

## 技术实现

### 语言管理
- 使用 `@StateObject` 管理语言状态
- 语言偏好保存在 UserDefaults
- 切换语言后界面自动更新

### 本地化字符串
- 所有文本集中在 `LocalizedStrings` 结构体
- 支持动态添加新语言
- 支持格式化字符串（如分数显示）

### UI 适配
- 支持 iPhone 和 iPad
- 支持深色模式
- 响应式布局

## 扩展指南

### 添加新语言
1. 在 `AppLanguage` 枚举中添加新语言
2. 在 `LocalizedStrings.allStrings` 中添加翻译
3. 自动出现在语言选择器中

### 添加新文本
1. 在所有语言的字典中添加相同的 key
2. 提供对应的翻译
3. 使用 `languageManager.localizedString("key")` 获取

## 示例代码

### 添加新语言（例如法语）
```swift
enum AppLanguage: String, CaseIterable {
    // ... 现有语言
    case french = "fr"
    
    var displayName: String {
        case .french: return "Français"
    }
    
    var flag: String {
        case .french: return "🇫🇷"
    }
}

// 在 LocalizedStrings.allStrings 中添加
.french: [
    "app_title": "Devine le Geste",
    "upgrade_to_pro": "Passer à Pro",
    // ... 其他翻译
]
```

### 添加新文本
```swift
// 在所有语言中添加
"new_feature": "新功能"  // 简体中文
"new_feature": "New Feature"  // 英语
"new_feature": "新機能"  // 日语
// ... 其他语言

// 使用
Text(languageManager.localizedString("new_feature"))
```
