### 1. 条款文字修改 ✅
**修改前**：
- 中文："条款适用。订阅将自动续费，除非取消。"
- 英文："Terms apply. Subscription automatically renews unless cancelled."

**修改后**：
- 中文："购买确认后将从您的iTunes账户扣费。订阅将自动续费，除非在当前订阅期结束前至少24小时取消。"
- 英文："Payment will be charged to your iTunes account at confirmation of purchase. Subscription automatically renews unless canceled at least 24 hours before the end of the current period."

### 2. 文字大小调整 ✅
**修改前**：
```swift

```

**修改后**：
```swift

```

### 3. 隐私政策和使用条款链接修复 ✅

**修改前**：
```swift
Button(languageManager.localizedString("privacy_policy")) { } // 空操作
Button(languageManager.localizedString("terms_of_use")) { }   // 空操作
```

**修改后**：
```swift
Button(languageManager.localizedString("privacy_policy")) { 
    showPrivacyPolicy = true  // 显示隐私政策页面
}
Button(languageManager.localizedString("terms_of_use")) { 
    showTermsOfUse = true     // 显示使用条款页面
}
```

## 📋 测试验证步骤

### 1. 条款文字验证
1. 打开应用
2. 点击任意锁定分类进入订阅页面
3. 查看"立即订阅"按钮下方的条款文字
4. 验证文字是否为新的英文版本且字体很小

### 2. 隐私政策链接验证
1. 在订阅页面点击"隐私政策"按钮
2. 验证是否弹出隐私政策页面
3. 检查页面内容是否完整显示
4. 点击关闭按钮验证是否正常关闭

### 3. 使用条款链接验证
1. 在订阅页面点击"使用条款"按钮
2. 验证是否弹出使用条款页面
3. 检查页面内容是否包含7个条款部分
4. 验证页面样式是否简洁（纯文本效果）

### 4. 多语言验证
1. 切换应用语言（如果支持）
2. 验证条款文字是否正确切换
3. 验证隐私政策和使用条款页面内容是否对应语言
