# GitHub 网页上传指南

## 🎯 目标
将隐私条款和技术支持文件上传到 https://github.com/seventiny666/guessme-support.git

## 📋 需要上传的文件
从本地 `docs/` 文件夹中的4个文件：
- `docs/index.html` → 上传为 `index.html`
- `docs/privacy.html` → 上传为 `privacy.html`
- `docs/terms.html` → 上传为 `terms.html`
- `docs/README.md` → 上传为 `README.md`

## 🚀 操作步骤

### 步骤1：访问仓库
1. 打开浏览器，访问：https://github.com/seventiny666/guessme-support
2. 确保你已经登录GitHub账号

### 步骤2：清空现有文件（如果有的话）
如果仓库中已有文件，需要先删除：
1. 点击每个文件名进入文件详情页
2. 点击右上角的垃圾桶图标（Delete this file）
3. 填写提交信息："删除旧文件"
4. 点击 "Commit changes"

### 步骤3：上传新文件
1. 在仓库主页点击 "Add file" → "Upload files"
2. 将以下4个文件拖拽到上传区域：
   - 从本地 `docs/index.html` 文件
   - 从本地 `docs/privacy.html` 文件
   - 从本地 `docs/terms.html` 文件
   - 从本地 `docs/README.md` 文件

### 步骤4：提交更改
1. 在页面底部的 "Commit changes" 区域
2. 提交信息填写：`上传英文版本支持页面`
3. 描述填写：
   ```
   - 技术支持页面 (index.html)
   - 隐私政策 (privacy.html)
   - 使用条款 (terms.html)
   - 项目说明 (README.md)
   
   邮箱已更新为: seventiny007@126.com
   所有内容已翻译为英文
   ```
4. 点击 "Commit changes"

### 步骤5：启用 GitHub Pages
1. 进入仓库设置：点击 "Settings" 标签
2. 滚动到 "Pages" 部分
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" (如果没有main，选择"master")
5. Folder 选择 "/ (root)"
6. 点击 "Save"
