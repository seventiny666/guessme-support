## 🚀 部署步骤

### 第1步：打开仓库
```
https://github.com/seventiny666/guessme-support
```

### 第2步：上传文件（重复4次）
1. 点击 "Add file" → "Create new file"
2. 输入文件名
3. 粘贴文件内容
4. 点击 "Commit changes"

### 第3步：启用 GitHub Pages
1. Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: "main"
4. Folder: "/"
5. Save

### 第4步：验证
等待 1-5 分钟，访问 URL 验证

### 第5步：配置 App Store Connect
填写隐私政策和技术支持 URL
