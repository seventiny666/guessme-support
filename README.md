# GuessMe - 你比划我猜

一款简单有趣的聚会游戏应用，适合家人朋友聚会时玩耍。一人比划一人猜，考验默契，增进感情！

## 📱 应用特性

- 🎲 **10大分类** - 500+精选词汇
- 📱 **多设备支持** - iPhone & iPad
- 🔒 **隐私保护** - 完全离线运行
- 🎯 **简单易用** - 无需学习，即开即玩
- 👨‍👩‍👧‍👦 **家庭共享** - 订阅支持家庭共享

## 🎮 如何开始

1. 在主界面选择一个分类
2. 选择游戏时长（30秒/60秒/90秒）
3. 点击"开始游戏"
4. 一人看屏幕比划，另一人猜词
5. 猜对点击✓，不会点击跳过
