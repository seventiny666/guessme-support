### 必须升级 Xcode

**当前版本：** Xcode 14.2 (iOS 16.2 SDK)  
**需要版本：** Xcode 16+ (iOS 18 SDK)

### 升级步骤

#### 1. 检查 macOS 版本
```bash
sw_vers
```
需要 macOS 14.5 (Sonoma) 或更高版本

#### 2. 升级 Xcode
**方式A：App Store（推荐）**
- 打开 Mac App Store
- 搜索 "Xcode"
- 点击"更新"或"获取"

**方式B：Apple Developer**
- 访问：https://developer.apple.com/download/
- 下载 Xcode 16+
- 安装到 Applications 文件夹

#### 3. 升级后重新打包
```bash
# 方式1：使用现有 Archive 重新导出
# 打开 Xcode -> Window -> Organizer
# 选择 Archive -> Distribute App

# 方式2：重新构建（推荐）
./快速打包脚本.sh
```
