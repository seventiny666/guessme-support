# 终端打包使用说明

## 前提条件

1. ✅ 已升级到 Xcode 16 或更高版本
2. ✅ 已登录 Apple Developer 账号（784430005@qq.com）
3. ✅ 证书和配置文件已配置好

## 方法一：快速打包（推荐）

### 使用快速脚本
```bash
./快速打包脚本.sh
```

这个脚本会：
1. 清理旧文件
2. 构建 Archive
3. 导出 IPA
4. 自动打开 Xcode Organizer

然后在 Organizer 中点击 "Distribute App" 上传。

## 方法二：完整打包流程

### 使用完整脚本
```bash
./build_and_upload.sh
```

这个脚本会：
1. 检查 Xcode 版本
2. 清理构建文件和缓存
3. 构建 Archive
4. 导出 IPA
5. 验证 IPA（可选）
6. 提供上传选项

## 方法三：手动命令

### 1. 清理项目
```bash
xcodebuild clean -project GuessMeIos.xcodeproj -scheme GuessMe -configuration Release
```

### 2. 构建 Archive
```bash
xcodebuild archive \
    -project GuessMeIos.xcodeproj \
    -scheme GuessMe \
    -configuration Release \
    -archivePath ./build/GuessMe.xcarchive \
    -destination "generic/platform=iOS" \
    CODE_SIGN_STYLE=Automatic \
    DEVELOPMENT_TEAM=346F2WR689 \
    -allowProvisioningUpdates
```

### 3. 导出 IPA
```bash
xcodebuild -exportArchive \
    -archivePath ./build/GuessMe.xcarchive \
    -exportPath ./build/export \
    -exportOptionsPlist exportOptions.plist \
    -allowProvisioningUpdates
```

### 4. 上传到 App Store Connect

#### 方式A：使用 Xcode Organizer（推荐）
```bash
open -a Xcode ./build/GuessMe.xcarchive
```
然后在 Organizer 中：
1. 选择 Archive
2. 点击 "Distribute App"
3. 选择 "App Store Connect"
4. 点击 "Upload"

#### 方式B：使用命令行（需要 API Key）
```bash
xcrun altool --upload-app \
    -f ./build/export/GuessMe.ipa \
    -t ios \
    --apiKey YOUR_API_KEY \
    --apiIssuer YOUR_ISSUER_ID
```

## 常见问题

### Q1: 提示 "Xcode 版本太低"
**解决方案：**
```bash
# 检查版本
xcodebuild -version

# 如果版本 < 16，需要升级 Xcode
# 参考 Xcode升级指南.md
```

### Q2: 签名错误
**解决方案：**
```bash
# 在 Xcode 中打开项目
open GuessMeIos.xcodeproj

# 检查签名设置：
# 1. 选择项目 -> GuessMe target
# 2. Signing & Capabilities
# 3. 确认 Team 是 346F2WR689
# 4. 确认 Bundle ID 是 com.seventinygame.app
```

### Q3: 构建失败
**解决方案：**
```bash
# 清理所有缓存
rm -rf ~/Library/Developer/Xcode/DerivedData/*
rm -rf ./build

# 重新构建
./快速打包脚本.sh
```

### Q4: 上传失败
**解决方案：**
1. 检查网络连接
2. 确认 Apple ID 已登录
3. 使用 Xcode Organizer 手动上传
4. 查看详细错误信息

## 验证构建

### 检查 Archive 信息
```bash
xcodebuild -exportArchive \
    -archivePath ./build/GuessMe.xcarchive \
    -exportPath ./build/info \
    -exportOptionsPlist exportOptions.plist \
    -exportArchive
```

### 查看 IPA 内容
```bash
unzip -l ./build/export/GuessMe.ipa
```

### 验证签名
```bash
codesign -dv --verbose=4 ./build/export/GuessMe.ipa
```

## 上传后的步骤

1. **登录 App Store Connect**
   - 访问：https://appstoreconnect.apple.com
   - 登录：784430005@qq.com

2. **查看构建状态**
   - 我的 App -> GuessMe
   - TestFlight -> iOS 构建版本
   - 等待"处理中"变为"可供测试"（通常 5-15 分钟）

3. **提交审核**
   - App Store -> 准备提交
   - 选择构建版本
   - 填写审核信息
   - 提交审核

## 时间估算

- 清理：5 秒
- 构建 Archive：2-5 分钟
- 导出 IPA：30 秒
- 上传：2-10 分钟（取决于网速）
- App Store 处理：5-15 分钟

**总计：约 10-30 分钟**

## 注意事项

1. ⚠️ 确保网络稳定，上传过程不要中断
2. ⚠️ 构建前先提交代码到 Git
3. ⚠️ 每次构建会自动增加 Build Number
4. ⚠️ 上传成功后不要立即删除 Archive
5. ⚠️ 保留 IPA 文件作为备份
