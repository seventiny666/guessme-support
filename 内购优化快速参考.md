## 📝 关键代码片段

### 1. 超时检测逻辑
```swift
// 创建超时任务（15秒）
let timeoutTask = Task {
    try await Task.sleep(nanoseconds: 15_000_000_000)
    if !Task.isCancelled {
        await MainActor.run {
            self.showTimeoutAlert = true
        }
    }
}

// 购买完成后取消超时任务
timeoutTask.cancel()
```

### 2. 加载动画组件
```swift
struct LoadingSpinner: View {
    @State private var isAnimating = false
    
    var body: some View {
        Circle()
            .trim(from: 0, to: 0.7)
            .stroke(Color.white, lineWidth: 3)
            .frame(width: 20, height: 20)
            .rotationEffect(Angle(degrees: isAnimating ? 360 : 0))
            .animation(
                Animation.linear(duration: 1)
                    .repeatForever(autoreverses: false),
                value: isAnimating
            )
            .onAppear { isAnimating = true }
    }
}
```

### 3. 使用加载动画
```swift
Button(action: { /* 购买逻辑 */ }) {
    HStack(spacing: 12) {
        if isPurchasing {
            LoadingSpinner()
            Text("处理中...")
        } else {
            Text("开始免费试用")
        }
    }
}
.disabled(isPurchasing)
```

### 4. 超时弹窗
```swift
.overlay(
    Group {
        if storeManager.showTimeoutAlert {
            TimeoutAlertView(
                message: storeManager.timeoutMessage,
                onRetry: { await storeManager.retryPurchase() },
                onDismiss: { storeManager.showTimeoutAlert = false }
            )
        }
    }
)
```

## 💡 最佳实践

1. **总是显示加载状态**
   - 让用户知道应用正在工作
   - 避免用户误认为卡死

2. **合理的超时时间**
   - 15秒是经过测试的最佳值
   - 太短会误判，太长影响体验

3. **提供重试选项**
   - 不要强制用户重启应用
   - 一键重试提升体验

4. **清晰的错误提示**
   - 告诉用户发生了什么
   - 提供解决方案

5. **优雅的状态管理**
   - 使用@Published自动更新UI
   - 避免状态不一致
