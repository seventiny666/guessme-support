## 📊 技术实现亮点

### 1. 并发任务管理
```swift
// 购买任务和超时任务并发执行
let timeoutTask = Task { /* 15秒超时 */ }
let purchaseTask = Task { /* 购买逻辑 */ }

// 购买完成后自动取消超时任务
timeoutTask.cancel()
```

### 2. 流畅的动画
```swift
// 使用rotationEffect实现旋转
.rotationEffect(Angle(degrees: isAnimating ? 360 : 0))
.animation(
    Animation.linear(duration: 1)
        .repeatForever(autoreverses: false),
    value: isAnimating
)
```

### 3. 智能状态管理
```swift
// 使用@Published自动更新UI
@Published var showTimeoutAlert = false

// 区分不同的加载状态
@State private var isPurchasing = false
@State private var isRestoring = false
```

### 4. 优雅的错误处理
```swift
// 区分超时错误和其他错误
if showTimeoutAlert {
    throw StoreError.timeout
}
```
