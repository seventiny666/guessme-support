# 快速修复订阅审核问题 - 5分钟解决

## 问题
App 审核被拒：订阅产品没有提交审核

## 立即执行（5步骤）

### 步骤1：访问订阅页面（1分钟）
```
https://appstoreconnect.apple.com/apps/6759666661/distribution/subscription-groups/21954022
```

### 步骤2：检查订阅产品状态（1分钟）
找到产品：`com.seventinygame.app.yearly`

**如果状态是"准备提交"：**
- 直接点击"提交以供审核"
- 跳到步骤5

**如果状态是"元数据丢失"：**
- 继续步骤3

### 步骤3：快速完善信息（2分钟）

#### 添加订阅本地化
点击订阅产品 -> 订阅本地化 -> 添加本地化

**简体中文：**
- 显示名称：`专业版年度订阅`
- 描述：`解锁全部10个分类和700+词汇，创建自定义分类，前3天免费试用，$1.99/年`

#### 添加审核说明
点击订阅产品 -> 审核信息

**审核说明：**
```
测试步骤：
1. 点击首页"升级到专业版"
2. 查看订阅详情
3. 使用沙盒账号购买
4. 验证专业版功能解锁
```

### 步骤4：提交订阅审核（30秒）
1. 点击"提交以供审核"
2. 确认提交

### 步骤5：重新提交App（30秒）
1. 返回 App 页面
2. App Store -> 准备提交
3. 重新提交审核

## 完成！
