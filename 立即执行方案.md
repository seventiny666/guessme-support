# 立即执行方案

## 🎯 目标
快速部署到 guessme-support 仓库

## ⚡ 最简单的方法

### 方法1：GitHub网页界面（推荐）

1. **打开仓库**: https://github.com/seventiny666/guessme-support

2. **上传4个文件**:
   - 点击 "Add file" → "Create new file"
   - 文件名: `index.html`
   - 内容: 复制 `docs/index.html` 全部内容
   - 提交
   
   重复上述步骤上传：
   - `privacy.html` (复制 `docs/privacy.html`)
   - `terms.html` (复制 `docs/terms.html`)  
   - `README.md` (复制 `docs/README.md`)

3. **启用GitHub Pages**:
   - Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/"
   - Save

4. **验证**: 1-5分钟后访问
   - https://seventiny666.github.io/guessme-support/

### 方法2：使用GitHub CLI（如果已安装）

```bash
# 克隆仓库
gh repo clone seventiny666/guessme-support
cd guessme-support

# 复制文件
cp ../iosguessme/docs/* .

# 提交推送
git add .
git commit -m "添加支持页面"
git push
```
