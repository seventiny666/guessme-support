# 内购流程图

## 1. 购买流程图

```
用户点击"开始免费试用"
         ↓
    设置加载状态
    isPurchasing = true
    isLoading = true
         ↓
    显示加载动画
    (旋转圆环 + "处理中...")
         ↓
    ┌─────────────────────┐
    │  启动两个并发任务    │
    └─────────────────────┘
         ↓
    ┌────────┴────────┐
    ↓                 ↓
购买任务          超时任务
    ↓                 ↓
发起购买请求      等待15秒
    ↓                 ↓
    ├─成功────→ 取消超时任务
    │              ↓
    │          完成购买
    │              ↓
    │          关闭弹窗
    │
    ├─用户取消──→ 取消超时任务
    │              ↓
    │          恢复按钮状态
    │
    ├─失败────→ 取消超时任务
    │              ↓
    │          显示错误提示
    │
    └─超时────→ 超时任务触发
                   ↓
              显示超时弹窗
                   ↓
              ┌────┴────┐
              ↓         ↓
          重新尝试    取消
              ↓         ↓
          重新发起    关闭弹窗
          购买请求    恢复状态
```

## 流程图说明

### 符号说明
- `→` : 流程方向
- `├─` : 分支
- `└─` : 最后分支
- `┌─┐` : 决策点
- `│` : 连接线

### 关键节点
1. **购买开始**: 用户点击按钮
2. **并发执行**: 购买任务和超时任务同时运行
3. **状态更新**: 通过@Published自动更新UI
4. **超时检测**: 15秒后自动触发
5. **用户选择**: 重试或取消
6. **资源清理**: 任务完成后自动清理

### 时间节点
- T0: 用户点击
- T0-T15: 加载状态
- T15: 超时检测
- T15+: 显示超时弹窗

### 状态转换
- 正常 → 加载 → 完成
- 正常 → 加载 → 超时 → 重试 → 加载
- 正常 → 加载 → 超时 → 取消 → 正常
